# LitAudio Initialkatalog (Mockdaten)

Diese Dateien enthalten den initialen Epochen-/Werkkatalog als Mockdaten für das MVP inkl. Reader-Erweiterung.

## Dateien
- `literary_periods.json`
- `authors.json`
- `works.json`
- `work_rights.json`
- `work_texts.json`
- `text_sections.json`
- `audio_editions.json`
- `audio_tracks.json`
- `reading_progress.json`
- `annotations.json`
- `bookmarks.json`

## Hinweise
- Alle Inhalte sind als `draft_mock` bzw. `mockup` markiert.
- `publish_allowed` ist initial `false`, bis Rechtsprüfung je Werk/Edition abgeschlossen ist.
- Kapitelstruktur ist für Audio + Reader aufeinander abgestimmt (über `audio_track_id`).
